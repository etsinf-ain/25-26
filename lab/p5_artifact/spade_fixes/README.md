Corrección de errores para que los artefactos funcionen con el servidor interno de XMPP de SPADE, pyjabber

Implementación para entornos virtuales con uv. Si usas otro gestor de paquetes, las rutas serán distintas.

Copia estas carpetas dentro de site-papckages de tu entorno virtual. Suele estar en 
.venv/lib/python3.12/site-packages/

Si vueelves a reinstalar los paquetes spade,, spade-artifiact o spade-pubsub se perderán estos cambios y tendrás que volver a hacerlo.
